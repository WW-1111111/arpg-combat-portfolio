# ARPG Combat Design Portfolio

This repository documents my self-directed learning journey 
toward becoming a combat designer in the Chinese game industry.

## Background

I am Wei Wu, an MSc Data Science and AI student at the 
University of Liverpool. My MSc dissertation focuses on 
LLM-based quest generation in action RPGs, supervised by 
Dr. Rasmus Ibsen-Jensen.

## Contents

- `/analysis` — Frame data analysis of action RPGs I have played
  - Sekiro: Shadows Die Twice
  - Black Myth: Wukong
  - Elden Ring
  - Elden Ring: Nightreign
- `/unity-experiments` — Small Unity prototypes for combat 
  systems (deflect, hitstop, dodge i-frames, boss AI)
- `/dissertation` — Code and notes related to my MSc 
  dissertation project

## Methodology

All frame data analysis is conducted using legal methods:
60fps gameplay recording with OBS, frame-by-frame inspection 
in DaVinci Resolve, and multiple-sample averaging. No game 
files are extracted, decompiled, or reverse-engineered.
